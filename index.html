<HTML> Common tag reference

Table of common tags

Tag Name Description Example
<H1> Heading Elements The <H1> to <H6> HTML elements represent six levels of section headings. <H1> is the highest section level and <H6> is the lowest. Click on example for further details.
<P> The Paragraph Element The <p> HTML element represent represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.
<a> The anchor element The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
